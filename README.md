# 🎯 CIMA Radio CRM - Sales Pipeline Management System

**Versión:** 2.0  
**Estado:** ✅ Production Ready  
**Última actualización:** 26 de Marzo 2025

---

## 📖 Descripción

Sistema de gestión de ventas (CRM) diseñado específicamente para el equipo de CIMA Radio. Permite gestionar clientes, iniciativas de venta y propuestas en un único lugar centralizado.

**No requiere instalación.** Solo abre el archivo en tu navegador.

---

## 🚀 Acceso Rápido

```
Abre en tu navegador: https://tuusuario.github.io/CIMA-Radio-CRM/
```

---

## ✨ Características Principales

- 👥 **Gestión de Clientes** - Clientes ilimitados con múltiples contactos
- 🎯 **Gestión de Iniciativas** - Crea paquetes de venta personalizados
- 📊 **Estados Personalizables** - Define tus propios estados con emojis
- 🎯 **Tablero Kanban** - Visualiza tu pipeline de ventas
- 📈 **Dashboard Analytics** - Métricas en tiempo real
- 📥 **Import/Export CSV** - Sincroniza datos fácilmente

---

## 💾 Datos

- ✅ Los datos se guardan **localmente en tu navegador** (localStorage)
- ✅ Cada usuario tiene su **propia copia de datos**
- ✅ Para compartir datos: **descargar CSV → importar en otro usuario**
- ⚠️ Si limpias cache del navegador, pierdes datos (haz backup con CSV)

---

## 📋 Campos de Contacto

Cada contacto incluye:
- Nombre y Apellido (separados)
- Email
- Teléfono
- Cargo/Posición (30 opciones estándar + personalización)
- Fecha de Nacimiento (opcional)

---

## 📊 Vistas Disponibles

1. **Tablero Kanban** - Columnas por estado
2. **Dashboard** - Métricas y gráficos
3. **Lista** - Vista plana de todas las propuestas
